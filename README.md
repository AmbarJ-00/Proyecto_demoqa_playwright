# Proyecto demoQa playwright

## Objetivo
Diseñar, implementar y ejecutar un conjunto de pruebas automatizadas end-to-end utilizando Playwright, con el fin de validar la funcionalidad, estabilidad y experiencia de usuario de los principales módulos del sitio DemoQA, garantizando que los flujos críticos se comporten correctamente en distintos navegadores y escenarios de uso.

## Descripcion
Este proyecto consiste en la creación de un framework de automatización de pruebas basado en Playwright, aplicado al sitio web DemoQA, una plataforma de demostración ampliamente utilizada para prácticas de testing.

Las pruebas están orientadas a validar componentes interactivos, formularios, elementos dinámicos y flujos funcionales clave, asegurando el correcto comportamiento de la aplicación bajo diferentes condiciones. El proyecto contempla la ejecución de pruebas en múltiples navegadores, la generación de reportes visuales, y la captura automática de evidencias como screenshots y videos ante fallos.

Además, se implementan buenas prácticas de QA como:
* Organización modular de los casos de prueba
* Manejo de datos de prueba
* Configuración centralizada mediante playwright.config
* Automatización reproducible y escalable

Este proyecto sirve como entorno de práctica profesional y como evidencia técnica del uso de Playwright para pruebas automatizadas en aplicaciones web modernas.
